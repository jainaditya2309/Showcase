Project Overview

MultiMantra is a comprehensive digital platform that combines multiple essential services into a unified application. It serves as an all-in-one solution for content creators, professionals, and individuals who need various digital tools for their online presence.


Core Philosophy: "One Mantra, Many Solutions" - MultiMantra aims to simplify digital life by providing multiple services under a single platform.

Tech Stack

Frontend Technologies

  • Next.js 15: React framework for production
  • React 19: JavaScript library for building UI
  • Tailwind CSS: Utility-first CSS framework
  • Responsive Design: Mobile-first approach
  • Modern JavaScript: ES6+ features
  • Component Architecture: Reusable UI components

Backend & Database

  • Node.js: JavaScript runtime environment
  • Express.js: Web application framework
  • MongoDB: NoSQL document database
  • NextAuth.js: Authentication library
  • API Routes: Server-side endpoints

Payment & AI Features

  • Razorpay: Payment gateway integration
  • Rule-based Chatbot: Customer support automation
  • Lucide React 0.503.0: Icon library
  • CORS 2.8.5: Cross-origin resource sharing

Installation & Setup

Prerequisites

  • Node.js 18.x or later
  • MongoDB database (local or Atlas)
  • Razorpay account (for payments)
  • GitHub OAuth app (for authentication)

Local Development Setup

# Clone the repository
git clone https://github.com/ADITYA9456/MultiMantra.git
cd MultiMantra

# Install dependencies
npm install

# Create environment file
cp .env.example .env.local

# Run the development server
npm run dev

# Open in browser
# Navigate to http://localhost:3000

Environment Variables

# MongoDB Connection
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/database

# NextAuth Configuration
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-secret-key

# GitHub OAuth
GITHUB_ID=your-github-client-id
GITHUB_SECRET=your-github-client-secret

# Razorpay (Optional)
KEY_ID=your-razorpay-key-id
KEY_SECRET=your-razorpay-secret-key

# Public URL
NEXT_PUBLIC_URL=http://localhost:3000

API Endpoints

Authentication

  • GET/POST /api/auth/[...nextauth]: NextAuth endpoints

Blog System

  • POST /api/blog: Create a new blog post
  • GET /api/blog?slug=<slug>: Get blog by slug

URL Shortener

  • POST /api/generate: Create shortened URL

Link Collector

  • POST /api/link: Create link collection

Payments

  • POST /api/razorpay: Process Razorpay payments

Database Schema

User Model

{
  email: String (required, unique),
  name: String,
  username: String (required),
  profilepic: String,
  coverpic: String,
  razorpayid: String,
  razorpaysecret: String,
  createdAt: Date,
  updatedAt: Date
}

Blog Model

{
  title: String (required),
  content: String (required),
  slug: String (required, unique),
  images: [String],
  author: ObjectId (ref: User),
  createdAt: Date,
  updatedAt: Date
}

Payment Model

{
  oid: String (required),
  amount: Number (required),
  to_user: String (required),
  name: String,
  message: String,
  done: Boolean (default: false),
  createdAt: Date
}

Authentication

NextAuth.js Configuration

  • Provider: GitHub OAuth
  • Session Strategy: JWT
  • Database Integration: MongoDB

Authentication Flow

  • User clicks "Login with GitHub"
  • Redirected to GitHub OAuth
  • User authorizes application
  • Callback creates/updates user in database
  • Session established with JWT

Protected Routes

  • /dashboard: User dashboard
  • /blog: Blog creation
  • /generate: Link collector

Payment Integration

Razorpay Setup

  • Account Creation: Sign up at Razorpay
  • API Keys: Get Key ID and Secret credentials
  • Webhook Configuration: Payment verification setup

Payment Flow

  • User initiates donation
  • Razorpay order created
  • Payment gateway opens
  • Payment processed
  • Webhook verifies payment
  • Database updated
