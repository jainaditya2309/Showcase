Project Overview

Chai Express is a modern web application built with Next.js. It provides users with the ability to create personalized donation pages where supporters can give them "chai donations". This is essentially a crowdfunding/donation platform that comes with a unique Indian chai theme.


Core Philosophy: "Support Through Chai" - The idea is to make small donations as common and social as sharing a cup of chai (tea) with someone.

Database Schema

User Model

{
  email: String (required),
  name: String,
  username: String (required, unique),
  profilepic: String,
  coverpic: String,
  razorpayid: String,
  razorpaysecret: String,
  createdAt: Date,
  updatedAt: Date
}

Payment Model

{
  name: String (required),
  to_user: String (required),
  oid: String (required), // Razorpay Order ID
  message: String,
  amount: Number (required),
  createdAt: Date,
  updatedAt: Date,
  done: Boolean (default: false)
}

Application Flow

1. User Registration/Login

  • User clicks "Login" ‚Üí Redirects to GitHub OAuth
  • GitHub authentication ‚Üí Receive user data
  • Check if user exists in database
  • If new user ‚Üí Create user record
  • User logged in ‚Üí Create session

2. Profile Setup

  • User navigates to Dashboard
  • Updates profile information
  • Save profile data to database
  • User profile page becomes active

3. Donation Process

  • Supporter visits user's donation page
  • Enters donation amount and message
  • Initiates payment via Razorpay
  • Payment process ‚Üí Webhook verification
  • Mark payment as complete in database

API Endpoints

Authentication

  • GET/POST /api/auth/[...nextauth]: NextAuth.js handlers
  • Handles GitHub OAuth authentication flow
  • Session management
  • User creation/authentication

Payments

  • POST /api/razorpay: Payment webhook
  • Verifies Razorpay payment signatures
  • Update payment status in database
  • Redirect to success page

Deployment Setup

Environment Variables Required

# NextAuth.js
NEXTAUTH_URL=your-domain.com
NEXTAUTH_SECRET=your-secret-key

# GitHub OAuth
GITHUB_ID=your-github-client-id
GITHUB_SECRET=your-github-client-secret

# Razorpay
KEY_ID=your-razorpay-key-id
KEY_SECRET=your-razorpay-key-secret

# Database
MONGODB_URI=mongodb://localhost:27017/Chai

# Application
NEXT_PUBLIC_URL=your-domain.com
